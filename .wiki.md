# Project Summary
FinnGuard Capital is a dynamic web application dedicated to connecting users with a range of lending partners for home, car, and personal loans. The platform emphasizes user engagement and lead conversion through features like an EMI calculator, loan eligibility assessments, and a comprehensive document checklist. With a management team well-versed in financial services, FinnGuard Capital strives to provide a user-friendly experience that caters to diverse financial needs. Recent enhancements focus on improving core functionalities, including real-time input validation, chat interface interactions, and visual designs to boost user engagement.

# Project Module Description
- **Homepage**: Showcases loan products and statistics, featuring functional Quick Eligibility buttons.
- **About Page**: Outlines FinnGuard Capital's mission and values.
- **EMI Calculator**: A tool for calculating monthly EMIs with real-time adjustments.
- **Contact Page**: Multi-step form for loan inquiries, now with updated map coordinates.
- **Testimonials**: Displays customer reviews in a carousel format.
- **FAQ Section**: Answers common loan-related questions.
- **Document Checklist**: Lists documents required for different loan types.
- **Live Chat Widget**: Provides real-time assistance with smart suggestions.
- **Eligibility Calculator**: Evaluates loan eligibility using AI-powered scoring.
- **Statistics Dashboard**: Visualizes key metrics with animated counters.
- **Advanced Lead Capture**: Multi-step form for lead qualification.
- **Conversion Optimizer**: Implements A/B testing to enhance conversion rates.
- **Advanced Calculator Suite**: Comprehensive tools for various loan calculations.
- **Content Marketing Platform**: Hosts blogs and educational resources.

# Technology Stack
- **Frontend**: React 19.1.1, TypeScript, Tailwind CSS, Shadcn-ui
- **Build Tool**: Vite 5.4.1
- **State Management**: React Query, Zustand
- **Routing**: React Router DOM 6.26.2
- **Animations**: Framer Motion 11.0.0
- **Linting**: ESLint
- **SEO Management**: react-helmet-async

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Run linting:
   ```
   pnpm run lint
   ```
3. Build the application:
   ```
   pnpm run build
   ```
