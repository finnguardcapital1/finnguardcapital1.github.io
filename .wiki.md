# Project Summary
FinnGuard Capital is a robust web application designed to connect users with various lending partners for home, car, and personal loans. The platform focuses on enhancing user engagement and improving lead conversion with features such as an EMI calculator, loan eligibility assessments, and a detailed document checklist. With a management team experienced in financial services, FinnGuard Capital aims to deliver a seamless user experience tailored to diverse financial needs. Recent developments have prioritized real-time input validation, an interactive chat interface, improved error handling, and visual enhancements to elevate user engagement.

# Project Module Description
- **Homepage**: Features loan products and a statistics dashboard, now with fully functional Quick Eligibility buttons.
- **About Page**: Details FinnGuard Capital's mission and values.
- **EMI Calculator**: Tool for calculating monthly EMI with real-time adjustments.
- **Contact Page**: Multi-step form for loan inquiries, integrated with updated Google Map coordinates.
- **Testimonials**: Displays customer reviews in a carousel format.
- **FAQ Section**: Provides answers to common loan-related questions.
- **Document Checklist**: Lists required documents for various loan types.
- **Live Chat Widget**: Offers real-time assistance with smart suggestions.
- **Eligibility Calculator**: Assesses loan eligibility with AI-powered scoring.
- **Statistics Dashboard**: Visualizes key metrics with animated counters.
- **Advanced Lead Capture**: Multi-step form for lead qualification.
- **Conversion Optimizer**: Implements A/B testing for conversion enhancement.
- **Advanced Calculator Suite**: Comprehensive tools for loan calculations.
- **Content Marketing Platform**: Hosts blog and educational resources.

# Technology Stack
- **Frontend**: React 19.1.1, TypeScript, Tailwind CSS, Shadcn-ui
- **Build Tool**: Vite 5.4.1
- **State Management**: React Query, Zustand
- **Routing**: React Router DOM 6.26.2
- **Animations**: Framer Motion 11.0.0
- **Linting**: ESLint
- **SEO Management**: react-helmet-async

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Run linting:
   ```
   pnpm run lint
   ```
3. Build the application:
   ```
   pnpm run build
   ```
