# Project Summary
FinnGuard Capital is a dynamic web application designed to connect users with various lending partners for home, car, and personal loans. The platform enhances user engagement and lead conversion through tools like an EMI calculator, loan eligibility assessments, and a comprehensive document checklist. With a management team experienced in financial services, FinnGuard Capital focuses on delivering a seamless, user-friendly experience tailored to diverse financial needs. Recent updates have refined core functionalities, such as real-time input validation and an interactive chat interface, to improve user interaction.

# Project Module Description
- **Homepage**: Showcases loan products and statistics with Quick Eligibility buttons.
- **About Page**: Provides insights into FinnGuard Capital's mission and values.
- **EMI Calculator**: Calculates monthly EMIs with real-time adjustments.
- **Contact Page**: Multi-step form for loan inquiries with updated map coordinates.
- **Testimonials**: Displays customer reviews in a carousel format.
- **FAQ Section**: Answers common loan-related questions.
- **Document Checklist**: Lists documents required for various loan types.
- **Live Chat Widget**: Offers real-time assistance with smart suggestions.
- **Statistics Dashboard**: Visual representation of key metrics with animated counters.
- **Advanced Lead Capture**: Multi-step form for qualifying leads.
- **Conversion Optimizer**: A/B testing framework to enhance conversion rates.
- **Advanced Calculator Suite**: Comprehensive tools for various loan calculations.
- **Content Marketing Platform**: Hosts blogs and educational resources.

# Technology Stack
- **Frontend**: React 19.1.1, TypeScript, Tailwind CSS, Shadcn-ui
- **Build Tool**: Vite 5.4.1
- **State Management**: React Query, Zustand
- **Routing**: React Router DOM 6.26.2
- **Animations**: Framer Motion 11.0.0
- **Linting**: ESLint
- **SEO Management**: react-helmet-async

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Run linting:
   ```
   pnpm run lint
   ```
3. Build the application:
   ```
   pnpm run build
   ```
