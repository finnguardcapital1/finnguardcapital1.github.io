# Project Summary
FinnGuard Capital is an innovative web platform designed to connect users with a variety of lending partners, offering tailored solutions for home, car, and personal loans. The application enhances user engagement and lead conversion through features like an EMI calculator, loan eligibility assessments, and a document checklist. With a seasoned management team in financial services, the platform aims to provide a seamless, user-friendly experience that caters to diverse financial needs. Recent updates include the removal of credit eligibility features and the transition to responsive product cards for improved adaptability across devices.

# Project Module Description
- **Homepage**: Displays loan products as responsive product cards with feature highlights.
- **About Page**: Provides insights into FinnGuard Capital's mission and values.
- **EMI Calculator**: Tool for calculating monthly EMIs with real-time adjustments.
- **Contact Page**: Multi-step form for loan inquiries with updated map coordinates.
- **Testimonials**: Showcases customer reviews.
- **FAQ Section**: Answers to common loan-related questions.
- **Document Checklist**: Lists documents required for various loan types.
- **Live Chat Widget**: Real-time assistance with smart suggestions.
- **Statistics Dashboard**: Visual representation of key metrics with animated counters.
- **Advanced Lead Capture**: Multi-step form for qualifying leads.
- **Conversion Optimizer**: A/B testing framework to enhance conversion rates.
- **Advanced Calculator Suite**: Comprehensive tools for various loan calculations.
- **Content Marketing Platform**: Hosts blogs and educational resources.

# Technology Stack
- **Frontend**: React 18, TypeScript, Tailwind CSS, Shadcn-ui
- **Build Tool**: Vite
- **State Management**: React Query, Zustand
- **Routing**: React Router DOM
- **Animations**: Framer Motion
- **Linting**: ESLint
- **SEO Management**: react-helmet-async

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Run linting:
   ```
   pnpm run lint
   ```
3. Build the application:
   ```
   pnpm run build
   ```
