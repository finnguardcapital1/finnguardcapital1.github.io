# Project Summary
FinnGuard Capital is an innovative web platform that connects users with a variety of lending partners, providing tailored solutions for home, car, and personal loans. The application enhances user engagement and lead conversion with features like an EMI calculator, loan eligibility assessments, and a detailed document checklist. With a seasoned management team in financial services, FinnGuard Capital is committed to offering a seamless, user-friendly experience that meets diverse financial needs. Recent enhancements include the removal of the credit eligibility features and the transition from an image carousel to responsive product cards, improving display adaptability across devices.

# Project Module Description
- **Homepage**: Displays loan products as responsive product cards with feature highlights.
- **About Page**: Offers insights into FinnGuard Capital's mission and values.
- **EMI Calculator**: A tool to calculate monthly EMIs with real-time adjustments.
- **Contact Page**: Multi-step form for loan inquiries with updated map coordinates.
- **Testimonials**: Showcases customer reviews.
- **FAQ Section**: Provides answers to common loan-related questions.
- **Document Checklist**: Lists documents required for various loan types.
- **Live Chat Widget**: Real-time assistance with smart suggestions.
- **Statistics Dashboard**: Visual representation of key metrics with animated counters.
- **Advanced Lead Capture**: Multi-step form for qualifying leads.
- **Conversion Optimizer**: A/B testing framework to enhance conversion rates.
- **Advanced Calculator Suite**: Comprehensive tools for various loan calculations.
- **Content Marketing Platform**: Hosts blogs and educational resources.

# Technology Stack
- **Frontend**: React 19.1.1, TypeScript, Tailwind CSS, Shadcn-ui
- **Build Tool**: Vite 5.4.1
- **State Management**: React Query, Zustand
- **Routing**: React Router DOM 6.26.2
- **Animations**: Framer Motion 11.0.0
- **Linting**: ESLint
- **SEO Management**: react-helmet-async

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Run linting:
   ```
   pnpm run lint
   ```
3. Build the application:
   ```
   pnpm run build
   ```
