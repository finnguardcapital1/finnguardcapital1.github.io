# Project Summary
FinnGuard Capital is a web application that connects users with various lending partners for home, car, and personal loans. The platform enhances user engagement and improves lead conversion through features like an EMI calculator, loan eligibility assessments, and a comprehensive document checklist. The management team, experienced in financial services, is committed to delivering a seamless user experience tailored to diverse financial needs. Recent updates have focused on real-time input validation, an interactive chat interface, improved error handling, and visual enhancements to boost user engagement. A critical fix was implemented for the Quick Eligibility button to ensure proper functionality.

# Project Module Description
- **Homepage**: Displays loan products and a statistics dashboard, now with functioning Quick Eligibility buttons.
- **About Page**: Outlines FinnGuard Capital's mission and values.
- **EMI Calculator**: Tool for calculating monthly EMI.
- **Contact Page**: Multi-step form for loan inquiries with updated Google Map coordinates.
- **Testimonials**: Showcases customer reviews.
- **FAQ Section**: Answers common loan-related questions.
- **Document Checklist**: Details required documents for various loan types.
- **Live Chat Widget**: Offers real-time assistance.
- **Eligibility Calculator**: Assesses loan eligibility.
- **Statistics Dashboard**: Displays key metrics.
- **Advanced Lead Capture**: Multi-step form for lead qualification.
- **Conversion Optimizer**: Implements A/B testing for conversion enhancement.
- **Advanced Calculator Suite**: Comprehensive tools for loan calculations.
- **Content Marketing Platform**: Hosts blog and educational resources.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Shadcn-ui
- **Build Tool**: Vite
- **State Management**: React Query
- **Linting**: ESLint
- **SEO Management**: react-helmet-async

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Run linting:
   ```
   pnpm run lint
   ```
3. Build the application:
   ```
   pnpm run build
   ```
